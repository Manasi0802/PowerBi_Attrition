📊 HR Analytics Projects

This repository contains HR Analytics projects focused on leveraging data to generate insights into workforce trends, employee behavior, and recruitment efficiency. These projects demonstrate the application of data analysis and visualization tools to support data-driven HR decision-making.

🔹 1. Employee Attrition Analysis & Dashboard
📌 Overview

This project analyzes employee attrition patterns and builds an interactive dashboard to monitor key HR metrics and identify factors influencing employee turnover.

🎯 Objectives
Analyze employee attrition trends across departments and demographics
Identify key drivers of employee turnover
Provide actionable insights to improve retention strategies
🛠 Tools & Technologies
Power BI
Excel
📊 Key Features
Attrition rate analysis 
Department-wise and age-group-wise attrition insights
Job satisfaction and gender-based trend analysis
Interactive dashboard with filters for dynamic exploration
📈 Key Insights
Identified high attrition in specific departments and age groups
Highlighted patterns between job satisfaction and employee turnover
Enabled data-driven decision-making for HR strategies
